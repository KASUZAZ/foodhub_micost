FOODHUB MICOST - STATIC WEBSITE

Projek ini tidak guna PHP, XAMPP, Apache, MySQL local, atau Supabase.
Ia dibina menggunakan teknologi asas sahaja:

1. HTML
2. CSS
3. JavaScript
4. localStorage browser untuk data demo ringkas

Fail utama:

1. index.html
2. assets/css/style.css
3. assets/js/app.js
4. assets/img/

Cara buka projek:

1. Buka folder foodhub_micost.
2. Double-click index.html.
3. Website akan terus dibuka dalam browser.

Cara deploy ke Netlify:

1. Buka https://app.netlify.com/drop
2. Drag folder foodhub_micost ke Netlify.
3. Netlify akan beri link website.

Fungsi yang ada:

1. Home page seperti contoh FoodHub.
2. Menu makanan dengan gambar.
3. Kategori makanan.
4. Search makanan.
5. Special deals.
6. Add to cart.
7. Order melalui WhatsApp.
8. Seller form untuk tambah menu demo.
9. Data menu tambahan disimpan dalam browser menggunakan localStorage.

Nota:

- Data demo localStorage hanya tersimpan di browser pengguna.
- Untuk projek beginner, ini lebih mudah diterangkan kerana tidak perlukan server.
- Jika mahu sistem sebenar dengan login dan database, barulah boleh tambah backend kemudian.
