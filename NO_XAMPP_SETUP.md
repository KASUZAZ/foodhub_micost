# FoodHub Tanpa PHP/XAMPP

Projek ini ialah static website. Maksudnya, ia boleh dibuka terus dalam browser tanpa install server.

## Teknologi Digunakan

- HTML untuk struktur laman
- CSS untuk reka bentuk laman
- JavaScript untuk menu, search, cart dan borang demo
- localStorage untuk simpan data demo dalam browser

## Cara Guna

1. Buka folder projek `foodhub_micost`.
2. Double-click `index.html`.
3. Website akan terus berjalan.

## Cara Deploy

Website ini boleh deploy ke:

- Netlify
- Vercel
- GitHub Pages

Untuk Netlify, drag-and-drop sahaja folder projek ke Netlify Drop.

## Kenapa Tidak Guna XAMPP/PHP?

Versi ini sesuai untuk projek beginner kerana:

- Tiada setup database.
- Tiada server local.
- Kod lebih senang dibaca.
- Sesuai untuk demo sistem FoodHub MICOST.

## Fungsi Demo

- Paparan Home seperti contoh FoodHub.
- Menu makanan bergambar.
- Filter kategori dan carian makanan.
- Special deals.
- Add to cart.
- Order melalui WhatsApp.
- Seller form untuk tambah makanan demo.

Nota: Data yang ditambah melalui seller form disimpan dalam browser pengguna sahaja.
